# SQL Server 알아보기 
  
  <sub>출처: [SQL Server Sample Database](https://www.sqlservertutorial.net/sql-server-sample-database/)</sub> 
  
  ```sql
  CREATE TABLE sales.stores (
          store_id INT IDENTITY (1, 1) PRIMARY KEY, 
          store_name VARCHAR (255) NOT NULL,
          phone VARCHAR (25),
          email VARCHAR (255),
          street VARCHAR (255),
          city VARCHAR (255),
          state VARCHAR (255),
          zip_code VARCHAR(5)
  );
  ```
